ナニミル

## サイト概要
風景や観光地のみなど(人物なし)の写真を投稿するもの

### サイトテーマ
旅先や地元の風景・景色などの写真を投稿する
他のユーザーからはコメントとしてその場所のレビューもできる

### テーマを選んだ理由
検索窓に場所名を打ち込んで調べ出すのではなく、写真を目で見て「ここへ行きたい！」と探せるものが欲しかったため

### ターゲットユーザ
これから旅行をする人へ
旅行が好きな人へ
自然や写真が好きな人へ

### 主な利用シーン
どこへ旅行へ行こうか考えているとき
自然を見て癒されたいなと感じているとき
